23755: Basic loader
The basic loader is set to auto-run from line 1. The POKE statement sets the border colour to blue, and the attribute for the lower part of the screen to 9 (blue INK, blue PAPER). Following this, the program loads a block of code of length 50 bytes to address 63000. This code block contains the rest of the loading routine, and is jumped to after it is loaded.
