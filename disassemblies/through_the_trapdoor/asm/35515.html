35515: Script Data: Berk Gaining Floating Power
The current position in Berk's script data is set to 35515 both when he gains the floating power (36536) and when the Clawed Foot on Level 3 grabs Berk (39391). In the former case, the script instruction at 35515 will be executed however in the latter case that instruction will be skipped over. Therefore, when Berk is grabbed on Level 3, the cycle attributes effect does not occur.
Input
Output
35515 DEFB 57,0 Cycle attributes (full-screen), clear display buffers and paint red areas outside current room
35517 DEFB 32,231 59168 (Floating / starting to fall, animated)
35519 DEFB 1,0,189,138 Jump to 35517
