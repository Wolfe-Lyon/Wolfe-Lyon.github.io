Read Cursor Shift byte. Bits 0-6 of this byte control where the cursor to write to Primary Display Buffer is moved to after writing the current character block. A value of 33 advances the cursor right by one character. A value, n, less than 33 moves the cursor down a character row, and left by 33-(n+1) characters.
