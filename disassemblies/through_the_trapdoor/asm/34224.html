34224: Remaining Duration of Berk's Current Power
For each game-cycle that a power is in use, the byte at 34224 is increased. When this reaches 16, it is reset to zero and the value at 34225 is decreased by one.
Input
Output
34224 DEFB 0 Number of elapsed game-cycles in current group of 16
34225 DEFB 0 Number of groups of 16 game-cycles remaining
