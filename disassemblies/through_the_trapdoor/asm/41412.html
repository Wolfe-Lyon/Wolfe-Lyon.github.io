41412: Table of Addresses of Attribute Data in Skeleton's Graphic Layout Data Blocks
Each of the addresses in this table is of the "operand" of a "set attribute" instruction (i.e. 242) in the Skeleton's graphic layout data blocks. These are initially 7 (i.e. white). When the skeleton is killed, the routine at 41424 (instructions starting at 41473) decreases this attribute by one per game cycle. When the attribute values reach 1 (blue), the skeleton is removed from the game and replaced by Boni.
