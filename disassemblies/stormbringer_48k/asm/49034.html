49034: Window data
Eight bytes per window. The meaning of the data is:
Offset Meaning
0 x-coordinate of left edge (characters)
1 x-coordinate of right edge (characters)
2 y-coordinate of top edge (characters)
3 y-coordinate of bottom edge (characters)
4 Border style
5 Border attribute
6 Initial (and minimum allowed) y-coordinate of hand cursor (characters)
7 Content attribute
The border style data refers to the index of the first window frame graphic element in a group (of eight) which are part of the character set at 27736. Character number 93, is the first element of the window frame group. This is followed by 101, 109, 117. 93 and 109 appear to be unused.
