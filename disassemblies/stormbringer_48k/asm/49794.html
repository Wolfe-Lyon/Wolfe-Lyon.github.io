49794: Table of 2x2 Block Style Definitions
Offset Meaning
0 Attribute
1 Terrain interaction parameter to apply
2 Terrain Interaction Parameter Painting Bitmap
3 Unused
4 Index of top-left udg
5 Index of top-right udg
6 Index of bottom-left udg
7 Index of bottom-right udg
The Terrain Interaction Parameter Painting Bitmap determines which of the four character cells spanned by a block have the relevant terrain interaction parameter applied. The bits in this value are:
Bit Effect
0 Apply to top-left cell
1 Apply to top-right cell
2 Apply to bottom-left cell
3 Apply to bottom-right cell
4 Unused
5 Unused
6 Unused
7 Unused
References brickwork graphics at 28504 and 29208.
