29272: Graphic lookup data
Table of addresses of start of graphic data for game graphics.
In some cases (marked "Graphics (standard)"), entries comprise of a pair of WORD values. The first of these is the address of the actual bitmap graphic data while the second value is the address of that graphic's mask bitmap data. The masks are used to allow, for example, background scenery to show behind objects that do not span the entire 2 × 2 character blocks when drawn. These standard graphics are generally for entities that can move, e.g. objects and characters, and are drawn by the routine at 63504.
In other cases (marked "Graphics (compacted)", entries comprise of a width value and a height value (both in characters) and an address for compacted graphic data. These compacted graphics are generally for static entities, such as scenery and terrain, and are drawn by the routine at 40664.
