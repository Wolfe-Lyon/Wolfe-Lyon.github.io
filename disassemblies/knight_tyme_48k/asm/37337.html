37337: Characters' initial stats
Nine groups of eight bytes, one group per character (including Magic Knight). The values are capped at 100:
Offset Effect
0 Strength (bits 0-6)
1 Happiness (bits 0-6)
2 Stamina (bits 0-6) / characters-can't-move flag (bit 7, Gordon only, see routine at 53731)
3 Charisma (bits 0-6) / can-help flag (bit 7)
4 Magic level / attribute (bits 0-6) / attribute FLASH flag (bit 7) (see trivia)
5 Character graphic index (if zero, then character is not drawn)
6 Flags
7 Index of current status text (e.g. "GORDON IS AT THE HELM CONTROL") (bits 0-3 only)
Characters' flags (byte 6) are as follows:
Bit Effect
0 Dislikes-Starfleet flag
1 Is-artificial flag (see trivia)
2 Unused
3 Unused
4 Unused
5 Unused
6 Can't-move flag (reset = character can move / set = character can't-move)
7 Asleep flag (reset = awake / set = asleep)
See also characters' current stats at 25020.
