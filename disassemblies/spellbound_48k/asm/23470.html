23470: Magic Knight movement flags
Bitwise flags which determine which direction(s) Magic Knight cannot currently move (i.e. set means can't move). The bits are as follows:
Bit Effect
0 Magic-Knight-can't-move-left
1 Magic-Knight-can't-move-right
2 Unused
3 Unused
4 Unused
5 Unused
6 Unused
7 Unused
23470 DEFB 0
