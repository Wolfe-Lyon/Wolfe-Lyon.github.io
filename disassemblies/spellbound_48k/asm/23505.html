23505: Animate-Magic-Knight's-puff-of-smoke-appearance-on-next-room-redraw flag
Determines whether or not Magic Knight appears in a puff of smoke the next time a room is (re)drawn. Only bit 0 is used.
Bit 0 Effect
Reset Puff of smoke is not animated
Set Puff of smoke is animated
