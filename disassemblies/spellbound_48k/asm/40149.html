40149: Common words table
This data block contains a list of zero-terminated (sub-)strings. Other text blocks reference this table when a particular string is required. This reduces the space requirements for storage of all of the game's text.
The game assigns indices to these strings, starting at 91. So, for example, if a text block contains a byte of value 95, that byte will be replaced with the sub-string of index 95 (5th entry) in this table.
